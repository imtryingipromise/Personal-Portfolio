# Personal Portfolio — Muqeet Mir

My personal portfolio website built with React, TypeScript, and Vite. Features a monochrome black & white design with interactive effects.

## Live Site

[mnmsportfolio.vercel.app](https://mnmsportfolio.vercel.app)

## Tech Stack

- React 18 + TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- Radix UI

## Features

- Interactive dot grid background with mouse tracking
- Floating particle field with cursor repulsion
- Text scramble animation on landing
- Scroll progress indicator
- Auto-scrolling tech marquee
- Animated border glow on hover
- Film grain noise overlay
- Dark / light mode toggle
- Responsive design
- Gmail compose integration for contact

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
